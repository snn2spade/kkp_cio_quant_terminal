# Features overview

- User Feature
    - To track current user and to load specific user data
    - No force log-in, there is guest mode to show non-private features/datas
    - Sign-up feature allow user to create user profile/ password without need to ask admin
    - Every sign-up default user role is "analyst"
    - available roles are "admin", "analyst"

- Analysts Feature/Page
    - To use as search portal, to find, to ask, to summarize research papers
    - be able to filter by houses, analysts, published date
    - compare lastest house views
    - view changes timeline for analyst

- Market Feature/Page
    - To show market headlines feed which aggregated from many sources
    - To show summarized latest insights or market talks

- Data Feature/Page
    - This page is focus on chart visualization
    - To visualize earning revision data as heatmap
    - To visualize consensus estimate data that allow to pick by region, sector
    - To visualize macro data
    - To visualize proprietary signals/indicators

- Signal Feature/Page
    - This page is tabular focus to show all active signals which current recomendation (buy, hold, sell)
    - each signal show some highlight stat e.g. hit rate

- Portfolio Feature/Page
    - This page is allow user to create portfolio using custom weight
    - To track portfolio performance

- Workflow Feature/Page
    - This page allow user to create custom workflow e.g. to send alert when porfolio is xxx
