# Project Name
CIO Quant Terminal

## Version
- Working: v0.3.0-dev
- Production: v0.2.0

# Project Overview
Modern full-stack single web application as a tool/terminal for quantititive analyst

The project is designed with:
- feature-based frontend structure
- scalable modular design/ reuseable react-components
- react component access data via backend apis
- no hidden crash message (user and dev friendly to deal with crash)
